# Atacadão - Nightmare

## Sobre

FPS 3D inspirado em clássicos como DOOM e QUAKE, escrito em C++, usando a API gráfica sokol_gfx, com foco em ter uma pequena game engine otimizada, tendo como objetivo rodar em multi-plataformas.
